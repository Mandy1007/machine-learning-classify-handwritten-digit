This folder contains Matlab codes for digit recognition written by Yunzhi Wang.
knn_query.m is the matlab function for implementing KNN
load_image_data.m and load_label_data.m are the codes for reading MNIST data
knn_rotate.m is the implementation of KNN with rotation pre-processing
MLP_PCA.m is the implementation of MLP with PCA pre-processing
SVM_linear.m is the implementation of linear SVM (implemented by not used in the project)
